This is a MATLAB implementation of Endmember estimation.

The method is presented in the paper: Simultaneously Counting and Extracting Endmembers in A Hyperspectral Image Based on Divergent Subsets

Run demo.m

data
A-Reference spectral
abf-Spectral abundance
c-Endemember number
mixed-Synthetic data
Nx,Ny,Nb-The size of Synthetic data
ind-Random number
